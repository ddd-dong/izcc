課程內容

C++

在這堂課你可以學到：
．輸入輸出
．迴圈
．條件判斷
．陣列……等基礎語法。
更進一步的還有實作課程，你可以親手寫出屬於自己的小專案，體驗到前所未有的成就感！
更何況，這門課也是高一資訊課的最佳前導，讓你事先熟悉相關概念，輕鬆應對未來的學習挑戰。不論你是完全零基礎還是已經有些程式基礎的學弟妹，都適合參加。

OpenCV

在這堂課你可以學到：
-OpenCV的基礎語法與功能
-電腦視覺的應用
OpenCV也常被應用於臉部辨識、手勢辨識、圖像分割等，是一個用途廣泛且功能強大的領域。
備註:雖然OpenCV可使用多種不同語言進行開發，但本課程將使用Python進行教學，需要使用 Python 程式知識（若遇到一些較難語法會再做複習），如語法、變數、函式等。

Python

在這堂課你可以學到：
- 輸入(input)、輸出(print)
- 註解(comment)
- 變數、變數型態轉換
- f-string
- 算術運算子...等基礎語法
並且透過一些基礎的實作，讓你輕鬆學會Python基礎語法。無論你是完全沒有編程經驗，或已有一些程式基礎，這門課都適合你。

Tkinter

在這堂課你可以學到：
-使用 Tkinter 建立各種視窗介面
-控制元件
-處理使用者輸入
-事件處理等相關技巧
Tkinter 功能豐富且易於學習，適合用於開發各種視窗應用程式，包括圖形化工具、資料視覺化、遊戲、教育軟體等。無論是初學者還是有經驗的開發者，都可以藉助 Tkinter 輕鬆建立吸引人的使用者介面。
備註:本課程需要使用 Python 程式知識(若遇到一些較難語法會再做複習），如語法、變數、函 式等。

AI

本課程將就人工智慧中幾個常見的領域，如強化學習、機器學習和深度學習等，進行簡單的介紹。並帶領學員使用機器學習訓練簡易的模型進行資料的預測。

HTML

這門課將介紹網頁的基本設計排版，如字體字型大小、放上圖片、連結等……。HTML是Hyper Text Markup Language 的簡稱，是一種建立於網頁的標準標記語言，易於學習又執行快速，常與CSS與JavaScript 一起被眾多網站用於設計網頁、網頁應用程式
